#  Sistema de Gestão Financeira Pessoal

Um sistema web simples para controle financeiro pessoal, desenvolvido em **React**.  
Permite registrar receitas e despesas, visualizar resumo financeiro e organizar lançamentos de forma prática.

---

##  Funcionalidades

- ✅ Adicionar receitas e despesas
- ✅ Listagem de lançamentos financeiros
- ✅ Resumo automático de saldo
- ✅ Interface moderna e responsiva
- ✅ Modo Dark/Light
- ✅ Layout centralizado estilo app bancário

---

##  Tecnologias utilizadas

- React
- JavaScript (ES6+)
- CSS3
- Vite
- Git & GitHub
